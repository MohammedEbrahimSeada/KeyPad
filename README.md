### Descirption

A personal KeyPad designed to easily use my computer in daily use with pretty design to put beside my PC, I would use it to access Copy, paste, Screenshot and Task Mgr.

- 4x Cherry MX Switches
- 4x SK6812 MINI Leds
- 1x XIAO RP2040
- 4x Blank DSA Keycaps
- 4x M3x16 Bolt
- 4x M3 Heatset
